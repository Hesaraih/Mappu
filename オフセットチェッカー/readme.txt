オフセットチェッカー
・オフセットをメモリからサーチしてみるツール

■利用は自己責任で

■仕様
・間違える可能性あり
・機械語のパターンからオフセットと思われる部分を抽出して表示
・VUにてパターンが変わらなければサーチ可能
・Mappu、箱産白魔、Bazaar、GraphicsID2Jobsに対応
・新たに倉庫番、Realtimeloggets、XIACE、FFXI小道具(一部)、Fish改に対応したつもり
・実行は現在立ち上がっているプロセスからオフセットをサーチ
・選択実行はdllを選択してオフセットサーチ
